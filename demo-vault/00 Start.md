Welcome to the [Sample Plugin Extended](https://github.com/mnaoumov/obsidian-sample-plugin-extended/) demo vault. This is a **sample/template plugin** - a more advanced version of the official [Obsidian Sample Plugin](https://github.com/obsidianmd/obsidian-sample-plugin/) that demonstrates the building blocks a real plugin uses: commands, a ribbon icon, a status bar item, modals, editor extensions, an editor suggest, side views, and a full settings tab. Everything below is a sample feature - the point is to show *how* each piece is wired, not to be useful on its own.

**How to try it:** open a feature note below and follow its steps. Most features surface as **notices** (the toasts in the top-right corner), so keep an eye there after you run a command or click the ribbon icon.

> [!TIP] Interactive buttons
>
> The two setup notes have **Run** buttons, powered by [`CodeScript Toolkit`](https://github.com/mnaoumov/obsidian-codescript-toolkit/), which this vault installs for you automatically on first open (see [[06 CodeScript Toolkit prerequisite]]). The sample features themselves are driven from the Obsidian UI (Command Palette, ribbon, editor), so the feature notes have no buttons - just steps to follow.
